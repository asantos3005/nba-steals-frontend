A simple frontend made in React to display a table of NBA Players based on their steals in the 25-26 season.

Data is from Kaggle - NBA Player Stats Dataset 2026:
https://www.kaggle.com/datasets/nilesh2042/nba-player-stats-2026

## Overview
The project was learning project to learn the fundamentals of Entity Framework Core in a fun way. I'm a big fan of the NBA so I thought using EF Core to handle data models of NBA stats would be a engaging way to learn.

Backend Repository: https://github.com/asantos3005/nba-steals-api

## Features
- Display TanStack table of NBA Player steal data
- Pagination
- Sorting
- Search by player or team name

## What I Learned
I learnt 2 main technologies in this project:

#### Entity Framework Core
- Learnt the key components to setup EF Core like the Database Context
- Created Migrations
- Learnt how to seed data from CSV

#### TanStack Table
- Learnt how to setup columns
- Learnt how to render rows
- Learnt about different row models to implement sorting and pagination
